> A full MLOps CI/CD pipeline for tomato crop growth stage detection using U-Net segmentation and CNN classification. Built with GitHub Actions — every push automatically lints, trains, tests, evaluates, and deploys the model.

---

## 📌 Pipeline Architecture

```
Developer Push → Setup Env → Lint → Unit Tests → Train Model → Evaluate → Upload Artifacts → Deploy
```

- **CI (Continuous Integration):** Setup → Lint → Tests → Train → Evaluate
- **CD (Continuous Delivery):** Upload artifacts → Deploy to Hugging Face Hub

---

## 🧠 Model Architecture

| Model | Role | Framework |
|---|---|---|
| U-Net | Leaf segmentation → Leaf Pixel Fraction | TensorFlow/Keras |
| ResNet50 | Growth stage classification | TensorFlow + ImageNet |
| DenseNet121 | Growth stage classification (comparison) | TensorFlow + ImageNet |
| InceptionV3 | Growth stage classification (comparison) | TensorFlow + ImageNet |
| VGG19 | Growth stage classification (comparison) | TensorFlow + ImageNet |

**Growth Stages (BBCH Scale):** Seedling → Vegetative → Flowering → Fruiting

## 🚀 Quick Start

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Prepare your dataset
Place your tomato images in the `data/` folder:
```
data/
├── Seedling/      (100+ images)
├── Vegetative/    (100+ images)
├── Flowering/     (100+ images)
└── Fruiting/      (100+ images)
```

### 4. Train the model
```bash
python src/train.py
```

### 5. Run tests
```bash
pytest tests/ -v
```

### 6. Predict on a new image
```bash
python src/predict.py path/to/tomato_image.jpg
```

## ⚙️ GitHub Actions Setup

1. Push this repo to GitHub
2. Add these secrets under **Settings → Secrets → Actions**:
   - `HF_TOKEN` — your Hugging Face API token
   - `HF_REPO_ID` — e.g. `your-username/tomato-growth-stage`
3. Every push to `main` triggers the full pipeline automatically

## 🔬 Research Background

This project is based on:
- **BBCH Scale** for tomato growth stage classification (Cardoso et al., 2021)
- **Leaf Pixel Fraction** for leaf density estimation from segmentation masks
- **Transfer Learning** with ResNet50, DenseNet121, InceptionV3, VGG19
- **U-Net** for pixel-level leaf segmentation with skip connections
